Fetcha

시놀로지에게 다운로드를 넘기는 가장 빠른 방법 · A fast remote for Synology Download Station

Fetcha는 iPhone에서 마그넷·토렌트·URL을 시놀로지 Download Station으로 바로 보내고, 진행 상황을 확인하는 비공식 서드파티 클라이언트입니다. 브라우저에서 마그넷 링크를 탭하면 곧바로 NAS가 받기 시작합니다.

주요 기능 / Features

  • 마그넷 링크 원탭 추가 (Safari·공유 시트) / One-tap magnet add
  • .torrent 파일 및 다운로드 URL 전송 / Send .torrent files and URLs
  • 진행률·속도·완료 확인, 완료 알림 / Progress, speed, and completion alerts
  • 토렌트 내 파일 선택 다운로드 / Per-file selection inside a torrent
  • QuickConnect 또는 로컬 IP 직접 접속 / QuickConnect or direct local IP

비밀번호는 안전한가요? 계정·비밀번호는 이 기기의 iOS Keychain에만 저장되며 사용자 본인의 NAS 로그인에만 쓰입니다. 외부로 전송되지 않습니다.

NAS가 없어도 볼 수 있나요? 앱 내 데모 모드로 실제 NAS 없이 화면을 둘러볼 수 있습니다.
